# USB-HARPOON

**USB-HARPOON** is a Linux-based USB security testing toolkit intended **only** for authorized, controlled lab environments.  
This repository provides safe, defensive tooling, documentation, and task notes for researchers and engineers verifying USB defenses and policies.

> ⚠️ **Legal & Ethical Notice:** Use this project only in environments where you have explicit permission. Misuse may be illegal. Read the documentation and obtain consent before testing.

---

## What this repo contains
This repo intentionally starts minimal — it currently provides only:
- Project description and safety rules
- Guidance for contributors and authorized researchers

If you want more tools or safe demos, check `TASK-NOTES.md` (coming soon) or request access.
